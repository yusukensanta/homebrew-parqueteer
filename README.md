# homebrew-parqueteer

Homebrew tap for [parqueteer](https://github.com/yusukensanta/parqueteer) — a CLI tool for reading, writing, inspecting, and converting Apache Parquet files across local storage, S3, GCS, and Azure without Spark.

## Install

```bash
brew tap yusukensanta/parqueteer
brew install parqueteer
```

## Usage

```bash
parqueteer --help
```

See the [parqueteer repository](https://github.com/yusukensanta/parqueteer) for full documentation, examples, and source code.
